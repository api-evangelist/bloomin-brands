# Bloomin' Brands (bloomin-brands)

Bloomin' Brands, Inc. (NASDAQ: BLMN) is one of the largest casual dining restaurant companies in the world, headquartered in Tampa, Florida and founded in 1988. It owns and operates four restaurant concepts — Outback Steakhouse, Carrabba's Italian Grill, Bonefish Grill, and Fleming's Prime Steakhouse & Wine Bar — across the United States and internationally (Brazil, South Korea, Hong Kong, and China). The company's consumer technology centers on the Web-based Dine Rewards loyalty program, per-brand mobile apps (iOS/Android) for online ordering, curbside takeaway, delivery, table payment, reservations, and waitlist, and corporate supplier portals (BBI Connect). Bloomin' Brands does not currently publish a public developer API, SDKs, or a developer portal.

## APIs

No public developer API is currently available from Bloomin' Brands. The company's digital surface is delivered through consumer mobile apps, a Web-based loyalty program (Dine Rewards), and corporate supplier/franchise portals rather than a documented, publicly accessible API. This repository tracks the company for the API Evangelist network; the `apis` collection in `apis.yml` is intentionally empty until a public API is published.
